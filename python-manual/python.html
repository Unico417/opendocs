프로그램의 기본

프로그램 순서

컴퓨터는 프로그램을 위에서 아래로 읽습니다.
당신은 프로그램을 위에서 아래로 순서대로 써야 합니다.
조심해서 쓰시죠.

메모

프로그램 안에 메모를 남길 수 있습니다.
#더 뒤에 적힌 것은 컴퓨터는 읽지 않습니다.
메모로 사용할 수 있습니다.

문자의 인자

print()는()안에 넣은 것을 표시합니다.

print('My name is Canola.')
# My name is Canola.

기본적인 계산

문자와 수치

프로그램에서는 수치는 그대로 기술합니다.
문자는'나"로 끼워야 합니다. 

# 수치 100
print(100)

# 문자 Canola
print('Canola')

문자에 '나"가 없으면 에러가 납니다.

print(Canola)
# NameError: name 'Canola' is not defined

프로그램 내에서 문자와 수치는 다른 것으로 취급됩니다.
자세한 내용은 수치 계산 항목에서 설명합니다.

변수

변수는 컴퓨터가 문자나 수치를 외우기 위해 필요한 것입니다.
변수에는 이름을 붙입니다.
갑자기 이름을 쓰기만 하면 프로그램은 변수라고 인식합니다.

변수 안에 데이터를 넣기 위해서는 이퀄=을 사용합니다.
이보다 오른쪽에 있는 계산 결과가 변수 안으로 들어갑니다.

print()의 ()안에 변수를 넣으면 그 내용물을 확인할 수 있습니다. 

name = 'Canola'
# 변수명 = 문자
# name 이라는 변수에 'Canola' 라는 문자가 들어갑니다.

age = 19
# 변수명 = 수치
# age 라는 변수에 19 라는 수치가 들어갑니다.

print(name)
# Canola
print(age)
# 19

수치 계산

=보다 오른쪽에 있는 계산 결과가 변수 안으로 들어갑니다.

연산 기호 목록
계산 기호 개요
+ 덧셈
- 뺄셈
* 곱셈
/ 나눗셈
% 나눗셈을 나눈 나머지
/ 제곱 
# 덧셈
answer = 15 + 7
print(answer)
# 22

# 뺄셈
answer = 15 - 7
print(answer)
# 8

# 곱셈
answer = 15 * 7
print(answer)
# 105

# 나눗셈
answer = 15 / 7
print(answer)
# 2.1428571

# 나눗셈을 나눈 나머지
# (나눗셈을 하여 나누지 못한 나머지 부분)
answer = 15 % 7
print(answer)
# 1

# 제곱
answer = 15 ** 7
print(answer)
# 170859375 ( = 15^7 )

변수 간의 계산

수치를 가진 변수끼리도 계산을 할 수 있습니다.

value1 = 100
value2 = 50
answer = value1 + value2

print(answer)
# 150

프로그램 내에서는 수치와 문자는 다르게 취급합니다.
문자의 '1'과 수치의 1은 다른 것이기 때문에 계산할 수 없습니다.
조심해서 다루죠. 

value1 = '100' # 문자
value2 = 50    # 수치

answer = value1 + value2
# TypeError: can only concatenate str (not "int") to str

계산 순서

계산 순서는 덧셈 뺄셈보다 먼저 곱셈 나눗셈이 이루어집니다.
곱셈 나눗셈보다 먼저 곱셈 계산이 이루어집니다.

이 기본적인 계산 순서는 ()를 사용하면 바꿀 수 있습니다.
()안에 있는 것이 우선적으로 계산됩니다. 

value1 = 15 / 3 + 2
value2 = 15 / (3 + 2)

print(value1)
# 7
print(value2)
# 3

Q2. BMI

우니코형는 키 173cm 몸무게 48kg입니다.
우니코형의 BMI를 구해주세요.

BMI = 체중[kg] ÷ ( 신장[m] )² 

width = 173
weight = 48
HINT 1

173cm는 센티미터이기 때문에 그대로 계산에 사용할 수 없습니다.
m로 변환해 보겠습니다.

HINT 1 의 쓰는 법 
meter = height / 100
정답

16.037...

height = 173
weight = 48

meter = 173 / 100

bmi = weight / (meter ** 2)
print(bmi)

리스트

변수의 형태 중 하나입니다.
수치나 문자 데이터를 나열한 것입니다.

리스트를 만들다

[]기호 안에 쉼표,로 구분하여 데이터를 기술합니다. []안에 아무것도 넣지 않으면 빈 리스트이 생깁니다. 

numbers = []
# 빈 리스트

numbers = [23, -591, 2.3, -32.9]

리스트의 내용물

x번째 데이터를 얻기 위해서는 list[x]와 같이 기술합니다.
프로그램 안에서는 순서는 0부터 시작합니다.
순서를 정하는 수치는 index라고 불립니다. 

numbers = [23, -591, 2.3, -32.9]
# index     0    1    2     3

member = ['Canola', 'Unico', 'Yuuki', 'Bit-kun']
# index      0         1        2         3

print(numbers[1])
# 591

print(member[0])
# Canola

index수치가 목록 길이보다 크면 오류가 납니다. 

numbers = []

print(numbers[1])
# IndexError: list index out of range
